# pancreas

GitHub-ready slim release of the pancreatic cancer metabolic cell death project.

## Included
- final and near-final result tables
- key figures for bulk, MR, single-cell, and drug modules
- scripts used to generate the analyses
- registries, manifests, stage summaries, and audit files
- metadata and project documentation

## Excluded
- raw data archives and sequencing files
- large binary objects and caches
- extracted raw-data working directories
- temporary, system, and debug files

## Manifests
- RESULTS_MANIFEST.tsv
- FIGURE_MANIFEST.tsv
- SCRIPTS_INDEX.tsv
- FILE_CLASSIFICATION.tsv

## Notes
This repository is intended for review, audit, and reproducibility of the core scientific outputs, not as a full raw-data archive.
