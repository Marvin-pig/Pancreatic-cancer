# Project structure

This slim repository keeps only the files needed to review the scientific workflow and final outputs.

## Core modules
- 01_metadata: study registry and metadata templates/audits
- 04_bulk_analysis: final bulk-seq results, figures, and validation outputs
- 05_MR: final MR registries, result tables, figures, and decision logs
- 06_scRNA: final single-cell registries, results, figures, and key logs
- 07_drug: final drug sensitivity outputs, registries, tables, figures, and logs
- 10_scripts: analysis scripts

## Generated helper files
- FILE_CLASSIFICATION.tsv: all source files classified as upload/exclude
- RESULTS_MANIFEST.tsv: uploaded tables and summaries
- FIGURE_MANIFEST.tsv: uploaded figures
- SCRIPTS_INDEX.tsv: uploaded scripts
