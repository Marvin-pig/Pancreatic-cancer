## 一、研究设计总览

### 核心逻辑链（六层递进）

```
第1层：泛癌景观扫描（广度）
    ↓ 锁定胰腺癌的独特特征
第2层：分子亚型鉴定 + 机器学习预后建模（深度）
    ↓ 筛出核心基因
第3层：孟德尔随机化因果验证（因果性）
    ↓ 确认哪些基因是"因"而非"果"
第4层：单细胞转录组解析（分辨率）
    ↓ 定位到具体细胞类型
第5层：免疫微环境 + 免疫治疗响应预测（临床价值）
    ↓ 连接免疫治疗
第6层：药物敏感性分析 + 药物重定位（转化潜力）
    ↓ 指向具体药物
```

## 三、分析模块详细流程

### 模块1：泛癌景观分析

**目的：** 展示三类代谢性细胞死亡在33种癌症中的全景图，并突出胰腺癌的独特特征。

**步骤：**

1. **表达分析**
   - 在TCGA 33种癌症中分析所有代谢性细胞死亡基因（~93个）的表达水平
   - 肿瘤 vs 正常 差异表达热图
   - 工具：`limma` R包
   - 输出：泛癌热图（Fig 1A）

2. **突变景观**
   - 分析这93个基因在TCGA-PAAD中的突变频率和类型
   - 工具：`maftools` R包
   - 输出：瀑布图（Fig 1B）

3. **拷贝数变异（CNV）**
   - 分析93个基因的CNV扩增/缺失频率
   - 数据：TCGA CNV数据（GISTIC2.0处理后）
   - 输出：CNV频率柱状图（Fig 1C）

4. **DNA甲基化**
   - 分析基因启动子区甲基化水平（β值）在肿瘤 vs 正常中的差异
   - 数据：TCGA 450K甲基化芯片
   - 输出：甲基化差异箱线图（Fig 1D）

5. **基因间相关性网络**
   - Pearson相关分析三类死亡基因间的共表达关系
   - 工具：`corrplot` R包
   - 输出：相关性网络图，展示ferroptosis-cuproptosis-disulfidptosis的交叉调控关系（Fig 1E）

**此模块的编辑加分逻辑：** 泛癌扫描 → 锁定PAAD的独特性 → 为后续深入分析提供合理性。

### 模块2：分子亚型鉴定 + 机器学习预后建模

**目的：** 基于代谢性细胞死亡基因的表达模式鉴定PAAD分子亚型，并构建预后评分模型。

**步骤：**

1. **一致性聚类（Consensus Clustering）**
   - 对TCGA-PAAD肿瘤样本，基于93个基因的表达数据进行无监督聚类
   - 工具：`ConsensusClusterPlus` R包
   - 参数：K=2-6, reps=1000, pItem=0.8, pFeature=1, clusterAlg="km"
   - 输出：累积分布函数（CDF）曲线 + 追踪图 → 确定最优K值（预期K=2或3）
   - 对各亚型进行生存分析（Kaplan-Meier + Log-rank）

2. **亚型间差异分析**
   - 临床特征差异（卡方检验 / Fisher精确检验）
   - 免疫浸润差异（ssGSEA + CIBERSORT）
   - 通路活性差异（GSVA）
   - 干性指数（mRNAsi）差异

3. **差异表达基因筛选**
   - 亚型间DEG分析（limma，|logFC| > 1, FDR < 0.05）
   - 与93个代谢性死亡基因取交集

4. **机器学习预后模型构建**

   **第一轮筛选：单因素Cox回归**
   - 对所有交集基因进行单因素Cox（P < 0.05保留）

   **第二轮筛选：多算法交叉验证**
   - LASSO Cox回归（10折交叉验证，选择lambda.min）
   - 随机森林（variable importance排序）
   - SVM-RFE（递归特征消除）
   - XGBoost（feature importance排序）
   - 取多算法共识基因（至少被2-3种方法选中的基因）

   **第三轮：多因素Cox回归**
   - 对共识基因进行多因素Cox → 获得最终模型基因和系数
   - 构建Risk Score = Σ(βi × Expi)

5. **模型评估**

   **内部验证（TCGA-PAAD，7:3分训练集/测试集）：**
   - Kaplan-Meier生存曲线（高/低风险组）
   - 时间依赖性ROC曲线（1年/3年/5年AUC）
   - C-index
   - 校准曲线

   **外部验证（≥2个GEO数据集 + ICGC队列）：**
   - 在GSE62452、GSE85916、ICGC-PACA中独立验证
   - 重复上述评估指标

6. **Nomogram构建**
   - 整合risk score + 年龄 + TNM分期 + 分化程度
   - 决策曲线分析（DCA）评估临床净获益
   - 工具：`rms` R包

**输出图表：** Fig 2（聚类）、Fig 3（模型构建过程）、Fig 4（Nomogram+验证）

### 模块3：孟德尔随机化因果验证 ⭐ 核心创新模块

**目的：** 验证模型中核心基因的表达/蛋白水平变化与胰腺癌风险之间是否存在因果关系。

**这一模块是区别于95%同类文章的关键差异化因素。**

**步骤：**

1. **eQTL-MR（基因表达水平 → 胰腺癌风险）**

   - **暴露端：** GTEx v8 胰腺组织eQTL数据
     - 对每个核心基因，选择其cis-eQTL（距转录起始位点1Mb内）作为工具变量
     - 筛选标准：P < 5×10⁻⁸，进行LD clumping（r² < 0.001, window=10000kb）
   - **结局端：** 胰腺癌GWAS汇总数据
     - PanScan+PanC4 meta-analysis（~8000 cases, ~13000 controls）
     - 从IEU OpenGWAS获取（ID: ieu-b-4959 或类似）
   - **分析方法：**
     - 主分析：逆方差加权法（IVW）
     - 敏感性分析：MR-Egger（检测方向性多效性）、加权中位数法、MR-PRESSO（异常值检测）
     - Cochran's Q检验（异质性）
     - MR-Egger截距（多效性）
     - Leave-one-out分析
   - **工具：** `TwoSampleMR` R包, `MendelianRandomization` R包

2. **pQTL-MR（蛋白水平 → 胰腺癌风险）** （如果数据可获取）

   - 暴露端：UK Biobank PPP 或 deCODE pQTL数据
   - 选择核心基因编码蛋白的cis-pQTL
   - 分析流程同上

3. **共定位分析（Colocalization）**
   - 对MR显著的基因-疾病对，进行共定位分析
   - 确认eQTL/pQTL信号和疾病GWAS信号是否源自同一因果变异
   - 工具：`coloc` R包 或 `HyPrColoc`
   - 判断标准：PP.H4 > 0.75（两个信号共享同一因果变异的后验概率）

4. **反向MR** （可选，增加说服力）
   - 暴露端：胰腺癌遗传易感性
   - 结局端：核心基因表达水平
   - 排除反向因果

**输出图表：** Fig 5（MR森林图 + 散点图 + Leave-one-out图）

### 模块4：单细胞转录组验证

**目的：** 在单细胞分辨率上验证核心基因的表达模式，定位到具体细胞类型。

**步骤：**

1. **数据下载与预处理**
   - 下载GSE155698（推荐，样本量最大）
   - 质控：过滤低质量细胞（nFeature_RNA > 200 & < 5000, percent.mt < 20%）
   - 标准化：SCTransform 或 NormalizeData + ScaleData
   - 降维：PCA → UMAP
   - 聚类：FindNeighbors + FindClusters
   - 注释：SingleR 或 手动marker基因注释

2. **核心基因表达分析**
   - 在UMAP上展示每个核心基因的表达分布（FeaturePlot）
   - 各细胞类型中的表达水平（DotPlot / VlnPlot）
   - 肿瘤细胞 vs 基质细胞 vs 免疫细胞的差异

3. **细胞间通讯分析**
   - 工具：CellChat
   - 分析核心基因参与的配体-受体对
   - 识别肿瘤细胞与免疫细胞间的关键通讯通路
   - 输出：通讯网络图、气泡图

4. **轨迹分析**（可选，增强深度）
   - 工具：Monocle3
   - 分析核心基因在肿瘤细胞分化轨迹中的动态变化

**工具：** Seurat (R) 或 Scanpy (Python)

**输出图表：** Fig 6（UMAP + FeaturePlot + DotPlot + CellChat）

### 模块5：免疫微环境与免疫治疗响应分析

**目的：** 探索risk score与免疫微环境特征及免疫治疗响应的关联。

**步骤：**

1. **免疫浸润分析（多算法交叉验证）**
   - CIBERSORT：22种免疫细胞比例
   - ssGSEA：28种免疫细胞富集评分
   - MCPcounter：8种免疫/基质细胞
   - ESTIMATE：免疫评分 + 基质评分 + 肿瘤纯度
   - 比较高/低风险组间的差异（Wilcoxon检验）

2. **免疫检查点分析**
   - 分析47个免疫检查点基因（PD-1, PD-L1, CTLA-4, TIM-3, LAG-3, TIGIT等）在高/低风险组的表达差异
   - 热图展示

3. **免疫治疗响应预测**
   - **TIDE算法**：预测免疫治疗响应（http://tide.dfci.harvard.edu/）
   - **IPS（Immunophenoscore）**：预测ICI疗效
   - **IMvigor210队列**（抗PD-L1治疗尿路上皮癌队列，可作为跨癌种验证）
   - 分析risk score能否区分响应者 vs 非响应者

4. **TMB（肿瘤突变负荷）分析**
   - 高/低风险组间TMB差异
   - TMB与risk score的相关性

**输出图表：** Fig 7（免疫浸润 + 检查点 + TIDE/IPS）

### 模块6：药物敏感性分析与药物重定位

**目的：** 评估高/低风险组对化疗/靶向药的敏感性差异，并基于核心基因进行药物重定位。

**步骤：**

1. **化疗药物敏感性分析**
   - 工具：`oncoPredict` R包（基于GDSC2数据库训练的模型）
   - 预测TCGA-PAAD每个样本对常用化疗药（吉西他滨、5-FU、奥沙利铂等）和靶向药的IC50
   - 比较高/低风险组的IC50差异

2. **CMap（Connectivity Map）药物重定位**
   - 输入：高/低风险组间的差异表达基因signature
   - 数据库：CMap (https://clue.io/)
   - 输出：能够逆转高风险特征的候选药物列表

3. **核心基因的druggability评估**
   - 查询DrugBank/DGIdb/ChEMBL
   - 对MR验证为因果基因且有已知药物的靶点，重点讨论

**输出图表：** Fig 8（药敏箱线图 + CMap结果）
